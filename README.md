## ✨ What Makes This Different?

Most portfolios are static pages. **VOID OS** is a living, breathing operating system in your browser:

- 🖥️ **It boots** — with a full BIOS-style diagnostic sequence
- 🌀 **It has a desktop** — a 3D holographic environment with orbiting cards around a breathing icosahedron
- 🧠 **It has an AI twin** — ask it anything about me, powered by Google Gemini
- 🎮 **It has Easter eggs** — hidden interactions and an achievement system
- 🔊 **It has sound** — a complete ambient soundscape engine
- ⌨️ **It has a command palette** — press `Ctrl+K` to navigate like a pro

## ⚡ Quick Start

```bash
# Clone the repository
git clone https://github.com/shivamsuhana/void-os.git

# Navigate to project
cd void-os

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your GEMINI_API_KEY in .env.local

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and experience the boot sequence.

## 🧩 Key Engineering Decisions

- **Single Source of Truth** — All portfolio data lives in one file (`portfolio-data.ts`). Add a project once, and it automatically updates across the Work section, Resume, Skills popup, and About stats.
- **Dynamic Imports** — Every section is lazily loaded with `next/dynamic` to keep the initial bundle minimal and boot sequence fast.
- **Custom Shader Pipeline** — Built a GLSL post-processing stack (bloom + chromatic aberration + vignette) for the 3D desktop instead of relying on heavy libraries.
- **Physics-Based Skill Graph** — Used D3's force simulation engine to create an interactive, force-directed neural map where skills are nodes and connections show relationships.
- **Zustand State** — Lightweight global state manages boot status, active section, cursor mode, sound preferences, and achievements across 25+ components.
